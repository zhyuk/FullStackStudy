#프로세스, 멀티태스킹, 스레드
- 프로세스(Process) : 실행중인 프로그램, 실행할 수 있도록 메모리에 올라와 있는 상태의 프로그램
- 멀티태스킹(Multi Tasking) : 여러 개의 작업을 동시에 처리할 수 있는 것
- 멀티프로세스 : 하나의 프로그램을 독립적으로 여러 작업을 동시에 실행할 수 있는 것
- 쓰레드(Thread) : 프로그램 안에서 실행되는 작업
- 멀티쓰레드 : 하나의 프로세스 안에 여러 개의 작업이 동시에 실행되는 것

멀티쓰레드가 이루어질 때 하나의 쓰레드에 이상이 발생되면 동일한 프로세스 안의 다른 쓰레드도 영향을 받는다.
하나의 프로세스 안의 쓰레드에 이상이 발생되어도 다른 프로세스는 영향을 받지 않는다.
