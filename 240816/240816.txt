#제이쿼리(jQuery)
    - 자바스크립트의 DOM호환성의 문제점을 보완하여 나온 기술
    - 자바스크립트 라이브러리의 종류 중 하나
    - 반드시 제이쿼리 라이브러리를 추가해야만 사용가능

    * 장점
        - DOM객체를 자유롭고 편하게 설정할 수 있다.
    * 단점
        - 필요없는(사용하지 않는) 기능까지 모두 추가를 해야한다.
        - 컴파일이 이루어져야 하므로 자바스크립트보다 시간이 오래 걸린다.

    - 1x 버전에만 모바일을 핸들링할 수 있는 함수가 포함되어 있음
    - 
        .min.js : 주석, 최소한의 공백 등 압축버전
        .slim.js : ajax, 애니메이션 등 포함되어있지 않은 경량버전
        .slim.min.js : 경량버전 + 압축버전
        .js : 일반버전

    [형식] 
        - $ 기호 => jQuery를 지칭하는 예약어 사용
            jQuery("선택자") 또는 $("선택자")로 표기
            jQuery(객체명) 또는 $(객체명)로 표기

    [권장사항]
        - 1개의 버전만을 적용

    * 라이브러리 추가 방식
        - 다운로드 방식 : 라이브러리 파일을 서버에 다운받아서 연결하는 방식
        - 네트워크 방식 : 라이브러리의 url로 연결하는 방식
    
    * 제이쿼리 기술 시 순서(유의사항)
        - 제이쿼리 라이브러리를 추가한 이후에 코드 기술
        - 로딩시점의 즉시실행 함수 안에 기술
        - html 엘리먼트가 전부 컴파일 된 이후에 기술

#즉시실행함수
    * 자바스크립트
        - (function(){...;})(); // 컴파일 시점에 실행. 가장 먼저 실행됨
        - window.onload = function(){...;} // 로딩 시점에 실행

    * 제이쿼리
        // 전부 로딩 시점에 실행된다.
        - $(function(){...;});
        - $(document).ready(function(){...;});

#제이쿼리에서 style 적용방법
    * 단일 속성 적용 시 : $("선택자").css("속성명", "속성값");
    * 다중 속성 적용 시 : **json 방식으로 기술** $("선택자").css({속성명: 값, 속성명: 값, ...});
