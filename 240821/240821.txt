# 이벤트 종류
    ** 이벤트 종류 : 이벤트함수의 매개변수에서 이벤트 type을 넣는 매개변수 값을 의미(문자열로 기술)

        * blur 속성
            이벤트 종류 : "blur"
            
            - js 이벤트핸들러 속성
            객체 또는 엘리먼트명.onblur = function(){...};
            객체 또는 엘리먼트명.addEventListener('blur', function(){});

            - jQuery이벤트핸들러 함수
            blur(function(){...});
            $(객체명).on('blur', function(){...});
            $(객체명).bind('blur', function(){...}); // 현재는 권장하지 않는 bind() 함수

# jQuery-ui 라이브러리
    : 전반적인 구조와 스타일, 기능등이 구현되어진 완전한 프레임워크를 제공해주는 자바스크립트 기반의 라이브러리를 말함.
        필요한 만큼 라이브러리를 추가해서 사용할 수 있다.
        반드시 jQuery 라이브러리를 먼저 추가한 후에 사용해야 한다.
