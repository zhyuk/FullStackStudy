OL 순서가 있는 목록 태그(ordered list)

        - width : 부모태그의 width 100% 인식, height : 컨텐츠 높이 만큼을 인식

        - 특수한 블록레벨 요소의 태그

        - 위, 아래 여백을 가지고 있다.

        - 마커 미 설정 시 기본값은 숫자값임.

        - li 태그를 자식태그로 가짐.


        

        ol태그

        - 다른 블록레벨 요소와 다르게 왼쪽 패딩을 가지고 있다.(마커영역 -40px정도)



        li태그

        - 다른 블록레벨 요소와 다르게 위, 아래 여백이 아예 없다.
