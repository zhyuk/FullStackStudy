ul 순서가 없는 목록 태그(unordered list)

        - width : 부모태그의 width 100%인식, height : 컨텐츠 높이 만큼을 인식
 
        - 특수한 블록레벨 요소의 태그

        - 위, 아래 여백을 가지고 있다.

        - 마커 미설정 시 기본값은 영역이 채워진 작은 점 모양임.

        - li태그를 자식태그로 가짐.

        

        ul태그

        - 다른 블럭레벨 요소와 다르게 왼쪽 패딩을 가지고 있다.(마커영역 -40px정도)



        li태그

        - 다른 블럭레벨 요소와 다르게 위,아래 여백이 아예 없다.
