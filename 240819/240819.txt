#CSS
    - href^=https : href 속성이 https로 시작되는 태그만 적용
    - href$=net : href 속성이 net으로 끝나는 태그만 적용
    - href*=google : href 속성에 google이 포함되는 태그만 적용


#제이쿼리에서 제공해주는 선택자
    * 탐색 선택자
        - :password : type=password인 태그만 적용됨
        - :text : type=text인 태그만 적용됨
        - :hidden : display=none인 태그만 적용됨. visibility:hidden은 인지할 수 없다.
        - :selected : 선택된 요소태그만 적용됨. -> 값이 변경될 때마다 인지하는 선택자
        - :contains(문자열) => 문자열 포함여부 확인 선택자
        - :has(요소명) => 해당 요소를 자식이나 자손으로 가지고 있는지의 여부 확인 선택자
        - :contents() => 자식(후손포함안됨)으로 존재하는 엘리먼트들
        - :not("선택자") => 해당 선택한 요소가 아닌 것
        - :end() => 바로 이전 선택 요소로 돌아가라는 기능을 제공해주는 함수. 현재 타겟의 요소를 종료한다는 의미

#URL
    * 파일경로
        file://드라이브명:₩폴더명₩...₩파일명(확장자포함)
    * 도메인경로
        http://도메인(ip주소,서버이름):포트번호/(컨텍스트-프로젝트명)페이지경로
        https://도메인(ip주소,서버이름):포트번호/(컨텍스트-프로젝트명)페이지경로
        
        기본포트를 이용할 경우 ':포트번호'는 생략될 수도 있다.
        컨텍스트를 없애는 이유 : 프로젝트명이 보이면 보안이 낮아지기 때문에 숨기는 경우가 많음.
