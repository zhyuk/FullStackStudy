배열(array)이란 :

        이름과 인덱스로 참조되는 정렬된 값의 집합
        - 배열값 : 배열 요소(element)
        - 배열에서의 위치: 인덱스(index) *0부터 시작
        - 배열 요소의 자료형이 달라도 된다.
        - 배열 요소의 인덱스가 연속적이지 않아도 되며, 특정 배열 요소가 비어있을 수도(empty) 있다.
        - 자바스크립트의 배열은 Array 객체로 다뤄진다.
        - 자료형은 object이다.

        *형식
        // 배열 맅너럴을 이용
        1. 선언키워드 객체명 = [배열요소1, 배열요소2, ...];
        // Array 객체의 생성자 이용
        2. 선언키워드 객체명 = Array(배열요소1, 배열요소2, ...);
        // new 연산자를 이용한 Array 객체 생성
        3. 선언키워드 객체명 = new Array(배열요소1, 배열요소2, ...);
