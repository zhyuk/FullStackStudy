#시간표기

GMT (그리니치 표준시간 : Greenwich Mean Time) : 1970년 1월 1일을 기준으로 영국 런던 지역의 시간을 표준시간대로 지정
UTC (세계 협정 시간 : Universial Time Coordinated) : GMT 시간대를 원자값 보정 및 윤초보정을 해서 나온 시간대

#예외(=익셉션 = Exception)
예측가능한 오류들을 처리해서 시스템이 비정상 종료가 되지 않도록 처리하는 것

- 나에게 발생한 에러를 내가 직접 처리하는 방식 : 
try{
	실행코드
} catch( 예외를 처리할 클래스로 만든 매개변수 ){
	try 블록 내에서 발생한 에러를 처리하는 코드
} [finally{
	try 블록 내에서 에러가 발생되든 발생안되든 반드시 처리해야하는 코드
}]
- 나에게 발생한 에러를 나를 호출한 곳에서 처리하도록 책임전가하는 방식: 메소드 선언문의 () 뒤에 throws를 이용하여 선언

[접근제어자] [기타제어자] 리턴타입 메소드명 ( [매개변수 선언문] ) throws 예외처리클래스명1, 예외처리클래스명2,... {...} {
	
}


Exception 클래스 : 자바에서 제공하는 최고 조상 예외 클래스
