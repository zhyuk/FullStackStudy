#jsp 액션태그
	<jsp:useBean id="객체명" class="패키지경로포함.클래스명까지"/> : 객체 생성 시 사용하는 액션태그
	<jsp:forward/> : 페이지를 포워드 방식(url 변경 X)으로 이동할 때 사용하는 액션태그

#jsp 내장객체 및 영역
	pageContext : 현재 페이지 내에서의 범위를 가지는 내장객체. 현재 페이지 내의 전반적인 정보를 담고 있고 페이지 내에서의 데이터를 설정하거나  변경할 때 사용하는 내장객체임.
	request : 클라이언트가 서버로 요청한 시점부터 서버가 클라이언트한테 응답하기 전까지 또는 요청을 처리할 때 리다이렉트를 만나기 전까지 살아있는 내장 객체임. **요청에 대한 전반적인 정보를(=데이터. 클라이언트ip, 세션아이디, url, 입력한 정보 등 모두) 담아가는 객체
	session : 현재 사용한 브라우저가 닫힐 때까지 또는 내가 세션 객체를 초기화하기 전까지 살아있는 내장 객체임. **대부분 로그인 정보를 처리할 때 사용하는 객체임. **세션객체 설정을 해주지 않으면 세션객체가 자동으로 소멸되는 시간은 30분임.
	application : 프로젝트가 실행되어 종료되는 순간까지 살아있는 내장객체. 애플리케이션의 전반적인 정보를 담고 있으며, 애플리케이션의 전반적인 설정이나 변경을 처리할 때 사용하는 객체이다.

#Parameter(파라미터) : get 방식이나 post 방식으로 전송하는 모든 데이터
